# ClaimRadar BG

Футуристична публична тестова версия за България: транскрипция на българско аудио/видео, откриване на проверими публични твърдения и първична онлайн проверка по надеждни източници.

## Версия 0.3

Добавено:

- нов режим **Реална проверка**;
- автоматично онлайн търсене по официални/надеждни източници;
- търсене по домейни като `nsi.bg`, `bnb.bg`, `nssi.bg`, `nra.bg`, `cik.bg`, `parliament.bg`, `ec.europa.eu`, `factcheck.bg`;
- индикативна оценка: намерени релевантни източници, първични следи за проверка, сигнал за спорно твърдение или нужна ръчна проверка;
- evidence cards с реални линкове;
- бутон **Реална проверка с източници** в текстовия режим;
- бутон **Транскрибирай и провери онлайн** в аудио/видео режима;
- запазен футуристичен cyber/neon дизайн от v0.2.

## Какво прави

- приема текст или `.txt` файл;
- приема аудио/видео файл;
- транскрибира чрез Faster Whisper;
- открива вероятно проверими твърдения;
- категоризира ги по теми;
- предлага надеждни български и европейски източници;
- опитва онлайн търсене по релевантни домейни;
- генерира `.srt` субтитри.

## Настройки

По подразбиране се използва:

```bash
WHISPER_MODEL_SIZE=base
WHISPER_DEVICE=cpu
WHISPER_COMPUTE_TYPE=int8
MAX_MEDIA_MB=80
SEARCH_TIMEOUT=8
```

За по-добро качество може да се зададе `small`, но на безплатен CPU ще е по-бавно.

## Дисклеймър

Това е тестов инструмент. Не е окончателна правна, политическа или журналистическа оценка. Всички резултати трябва да се проверяват по посочените източници.
